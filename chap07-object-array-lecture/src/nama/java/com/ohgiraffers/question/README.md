# 사용자 회원가입 및 로그인 시스템

- 회원 가입은 10명까지
- 가입한 회원이 없는 상태에서 로그인을 시도하면 가입한 회원이 없습니다. 출력
- 정상적으로 가입 후 로그인시 로그인 완료 출력되고 프로그램 종료
- 서비스에서 아이디, 이름, 비밀번호, 비밀번호 확인을 물어본 후
- 비밀번호와 비밀번호 확인이 같을 시에만 가입 진행

## 클래스 목록
- Application - 실행부
- LoginService - 화원가입, 로그인 메소드 처리
- LoginRepository - 회원 가입 처리, 로그인 처리
- User - DTO 클래스
